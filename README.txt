============================================================
PHONÉTIQUE DU FRANÇAIS — COMPONENTES DE LANDING PAGE ESTÁTICA
============================================================
Método del Prof. Sergio Tomassi
Desarrollado en HTML5, CSS3 y JavaScript interactivo (con Tailwind CSS CDN).

Estructura de archivos:
-----------------------
- index.html       : El documento principal con el marcado HTML completo.
- css/style.css    : Estilos personalizados y animaciones del ecualizador.
- js/script.js     : Interactividad y lógica del reproductor de video con subtítulos,
                     el menú móvil y la persistencia de datos del formulario en localStorage.

Cómo utilizar localmente:
-------------------------
Simplemente haz doble clic en 'index.html' o abre este archivo en cualquier navegador
web moderno para visualizar la landing page interactiva con total fidelidad.

Características del diseño y comportamiento:
--------------------------------------------
1. DISEÑO RESPONSIVO PREMIUM: Diseñado con una cuadrícula editorial sofisticada 
   que se adapta automáticamente a computadoras, tabletas y teléfonos móviles.
2. REPRODUCTOR DE VIDEO INTERACTIVO: Simulación interactiva con un ecualizador 
   dinámico en tiempo real y subtitulado bilingüe (francés/español) sincronizado.
3. MENÚ DE NAVEGACIÓN MÓVIL: Panel desplegable fluido y accesible para pantallas táctiles.
4. DIAGNÓSTICO PERSISTENTE: El formulario guarda de forma segura los envíos en el
   almacenamiento local del navegador (localStorage) para evitar pérdidas de progreso.
